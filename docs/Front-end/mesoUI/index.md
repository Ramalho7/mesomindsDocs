# mesoUI

A **mesoUI** é a biblioteca oficial de componentes da plataforma Mesominds, usada para **compartilhar UI entre múltiplos projetos**, incluindo:

- **Public site**
- **Dashboard administrativo**
- **Aplicações internas**

Ela garante consistência visual, padronização e reuso de código entre todos os fronts.

---

## Tecnologias

A biblioteca é construída com:

- **React**
- **TypeScript**
- **Vite**
- **Storybook**
- **Cypress**
- **Vitest**
