# Desenvolvimento front-end

No desenvolvimento Front-end será usado React com TS, e isso será integrado ao Back-End em Laravel, que serve todas as ações do sistema.

## Ideias iniciais

- Autenticação com JWT

## Requisitos não funcionais gerais

1. Sistema com alta acessibilidade devido ser um site de aprendizado
2. Design responsivo para suportar diferentes dispositivos e tamanhos de tela.
3. Código modular e reutilizável para facilitar a manutenção e evolução do sistema.
4. Compatibilidade com os principais navegadores (Chrome, Firefox, Edge, Safari).
7. Implementação de testes automatizados para garantir a qualidade do código.
8. Suporte a múltiplos idiomas (internacionalização).
9. Conformidade com padrões de segurança, como proteção contra XSS e CSRF.
