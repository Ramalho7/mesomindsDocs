# Back-end

Seção dedicada a apresentação área de back-end do projeto.

A codificação do back-end ocorre no repo: [Repo back-end](https://github.com/Ramalho7/mesomindsAPI)

O back-end será desenvolvido em Laravel 12.

Pré-requisitos para conseguir rodar o projeto:

1. PHP 8.3
2. Composer instalado
3. Extensões do PHP: ctype, curl, dom, fileinfo, mbstring, openssl, pcre, pdo, tokenizer, xml, zip
4. Servidor web(apache) ou docker

## Sistema freemium
O sistema deverá contar, em algum momento com o sistema freemium para disponibilizarmos conteúdos/simulados/provas/etc pagos e grátis.
