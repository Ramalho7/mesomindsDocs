# Questões

Questões ficaram disponíveis no sistema, como banco de questões para realização pelos usuários

## Requisitos

1. Questões podem conter alternativas infinitas
2. Questões podem possuir mais de uma questão correta
3. Questões podem ser abertas
4. Questões podem ser adicionadas imagens
5. Ediçao com LaTex ou algo similar
6. Possuir uma ou mais disciplinas relacionadas
7. Possuir uma ou mais materias relacionadas
8. Um aluno poderá ter o histórico das questões respondidas
