# Conteúdos

Conteúdos serão: publicações na plataforma, questões, simulados, etc.
Conteúdos serão adicionados pela área de ADMIN da plataforma, onde será possível escrever conteúdos de todos os tipos para servir a plataforma.

## Requisitos

1. Permitir Edição de conteúdo com LaTex
2. Permitir adição de código de linguagem de programação 
3. Permitir adicionar, remover, vizualizar (área de admin e publico), e editar conteúdos da plataforma
4. Histórico do user que adicionou/editou
5. Última edição com data
6. Data de criação
