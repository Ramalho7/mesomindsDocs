# Área do admin

Admin(dashboard adm) será a seção onde será gerenciado a criação de conteúdos da plataforma, excluir usuários indesejados da plataforma, etc.
Ele será responsável por gerir a plataforma por completo.

Possuirá alguns níves dentro desse dashboard, como: admin(super usuário), gerente, moderador, operador(criação de conteúdos, etc)

## Requisitos

1. Adição de conteúdos
2. Separação de responsabilidades por perfil nessa seção
3. Gerir/criar/remover/editar/visualizar conteúdos na plataforma
4. Hist. de conteúdos, adicionados/editados por usuário
