# Provas

- Provas serão criadas pelo professores para a turma e poderam ser feitas pelos alunos.

## Requisitos

1. Professor deve conseguir corrigir e adicionar nota as provas para os alunos.
2. Notas em provas só poderma ser alteradas após 8h depois que foram enviadas para o sistema
3. Provas devem conter questões que já existem na platforma(geral)
    - Também deve ser preciso que o professor possa criar questões próprias para as provas, essas questões não devem ir para o sistema e serão usadas apenas dentro da prova
4. Provas devem conter um histórico, todas as provas devem ficar visível para o professor 
5. Quando uma prova for adicionada, notificar o aluno
6. Quando a nota a prova for adicionada, notificar o aluno
