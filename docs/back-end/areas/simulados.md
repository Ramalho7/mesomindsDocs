# Simulados

Simulados serão uma junção de questões, como um ENEM.

## Requisitos

1. Simulado deve possuir um timer definido pelo usuário, quando finalizado o simulado é finalizado
2. Possuir calculo de nota
3. Possuir uma ou mais máterias/disciplinas, por meio das questões que estão nele
4. Histórico de simulados realizados para os alunos
5. possibilidade de refazer o simulado, porém mantendo o histórico do que foi feito anteriormente
