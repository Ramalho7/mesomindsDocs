Essa versão foi pensada para atender a algumas demandas:

1. Maior compromentimento com a ABNCC
2. Melhor relacionamento entre as entidades
3. Maior descrição e desacoplamento de responsabilidades em entidades

## Mudanças:

### Novas entidades: 

`Skill`

Alterada a entidade `Subject` para a adição da entidade `Skill` que representará de forma DIRETA uma habilidade de ABNCC.

Server apenas como uma tabela para "controle", a entidade a se relacionar com `Question` e `Content` o qualquer outra entidade é a `Topic`, permitindo uma hierarquia: `Topic` -> `SkillTopicPivot` -> `Skill` -> `Discipline` para melhorar relatórios, etc92q

`Discipline`

Representa as disciplinas escolar (Matemática, português, etc), servindo como entidade central para melhor gerenciamento pedagógico, por meio de entidades como `Skill` e `Topic`

`UserClass`

Tabela intermediária entre `SystemUser` e `Class`.

`QuestionCollectionClassPivot`

Tabela intermediária entre `Class` e `QuestionCollection`.
