## Seção Admin

A seção Admin é dedicada ao gerenciamento avançado da plataforma, permitindo que usuários autorizados (administradores) controlem conteúdos, questões, atividades, simulados, usuários e turmas.

Esta área é separada do site principal e acessível apenas para administradores autenticados.

### Funcionalidades disponíveis

- Gerenciamento de conteúdos
- Gerenciamento de questões
- Gerenciamento de atividades
- Gerenciamento de simulados
- Gerenciamento de usuários
- Gerenciamento de turmas
