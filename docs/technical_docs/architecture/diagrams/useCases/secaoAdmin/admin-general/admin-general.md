## Descrição

Este diagrama apresenta os casos de uso de gerenciamento da plataforma Mesominds, com foco na hierarquia de atores e suas permissões:

- **Admin**: Acesso completo a todas as funcionalidades
- **Moderador**: Acesso limitado a gerenciamento
- **Operador**: Acesso restrito a gerenciamento

## Casos de Uso Inclusos

- Gerenciar conteúdos
- Gerenciar questões
- Gerenciar atividades
- Gerenciar simulados
- Gerenciar usuários
- Gerenciar turmas
